FlowPlace: Flow Matching for Chip Placement

Peng Xie, Ke Xue, Yunqi Shi, Ruo-Tong Chen, Chengrui Gao, Siyuan Xu, Chenjian Ding, Mingxuan Yuan, Chao Qian

将流匹配（Flow Matching）生成模型引入芯片布局。针对现有扩散模型的三大痛点——合成数据预训练局限、采样慢、模块重叠——提出：(1) 掩码引导合成数据生成；(2) 灵活先验注入的流式高效训练；(3) 硬约束采样保证零重叠布局。在 OpenROAD 和 ICCAD 2015 基准上，PPA 优于 SOTA，采样效率提升 10-50×，且无模块重叠。

arXiv:2604.23602 · cs.AR / cs.LG · 2026-04-26

TimingLLM: A Two-Stage Retrieval-Augmented Framework for Pre-Synthesis Timing Prediction from Verilog

Armin Abdollahi, Negin Ashrafi, Mehdi Kamal, Massoud Pedram

提出 TimingLLM，两阶段 RAG-LLM 流水线，直接从 Verilog 代码预测综合后时序指标（WNS/TNS），无需运行 EDA 工具。Stage-1 微调 LLM 生成路径级时序特征；Stage-2 LLM 回归器结合 k-NN 检索库的对角引导向量预测 WNS/TNS。VerilogEval 上 WNS R²=0.91，TNS R²=0.97，速度比现有方法快 1.3-1.6×。同时发布 6 万模块 Verilog+综合报告数据集。

arXiv:2604.23100 · cs.CR / cs.AR / cs.LO · 2026-04-25

From Language to Logic: Bridging LLMs & Formal Representations for RTL Assertion Generation

Nowfel Mashnoor, Hadi Kamali, Kimia Azar

提出 ProofLoop，工具增强 ReAct 智能体，采用 solver-in-the-loop 方式从自然语言规范生成 SystemVerilog Assertions（SVA）。两阶段：Stage-A 通过 AST 向量库语义搜索 + JasperGold 结构查询自主收集设计上下文；Stage-B 生成 SVA 并用 JasperGold 形式证明反馈迭代优化（最多 3 轮）。FVEval Design2SVA 基准上语法正确率 93.7%、功能正确率 82.0%。

arXiv:2604.24649 · cs.AR · 2026-04-28

Déjà Vu Packing: Optimizing FPGA Logic Clustering Runtime via Pattern Memoization

Milo Liebster, Amin Mohaghegh, Andrew Boutros

FPGA 物理综合中 Logic Clustering（Logic Packing）是运行时主要瓶颈之一。本文提出模式记忆化（Pattern Memoization）方法，通过识别已处理的相同 LUT 聚类模式，跳过重复计算，显著缩短打包阶段运行时间，同时保持等价的布局质量。面向 VPR/VTR 框架的 FPGA 综合加速，具备工程落地价值。
