RuC: A Benchmark for HDL-Agnostic Rule Completion in Hardware Description Languages

Yao Lu, Hao Guo, Jianwu Dang, Jian Lu, Chen Luo, Yuetian Luo, Wuyue Liang, Zhao Zhang, Shu Chen, Dan Li

提出首个语言无关 HDL 规则补全基准 RuC，涵盖 Verilog/VHDL/SystemVerilog 三种硬件描述语言，评测 LLM 对 HDL 语法规则的理解与补全能力。构建了包含 HDL 典型规则的题库（缺失端口声明、时序逻辑错误、信号位宽不匹配等），通过 Pass@k 与准确率双维度评估 GPT-4o、Claude 3.7、DeepSeek-V3 等主流模型，揭示当前 LLM 在 HDL 规则推理上的关键瓶颈。

HAVEN: Automated UVM Testbench Synthesis with Large Language Models

Fikret Emin Akçelik, Emre Doğan, Sven Thümmel, Stefan Wildermann, Jürgen Teich

提出 HAVEN 框架，基于 LLM 自动合成符合 UVM 规范的验证测试台。利用结构化提示工程引导 LLM 生成 UVM 组件（driver/monitor/scoreboard/agent），实现模块间一致性约束。在 ISCAS-85/89 基准电路上评估，100% 编译成功率，覆盖率驱动测试台生成比人工编写代码减少 65% 工作量。有效解决 UVM 样板代码繁复的痛点。

SafeTune: Mitigating Data Poisoning Attacks in RTL Code Generation via GNN and XGBoost

Animesh Dey, Sayak Ray

针对 LLM 辅助 RTL 生成中的数据投毒安全威胁，提出 SafeTune 防御框架。利用图神经网络（GNN）提取 RTL 代码的图结构特征（AST/DFG/CFG），结合 XGBoost 分类器识别恶意植入的后门代码。在合成数据集上检测准确率超 97%，误报率低于 2%。首次将 GNN+ML 组合方法应用于 HDL 代码安全检测。

RKHS: RAG-Enhanced Kernel-based Heuristic Synthesis for High-Level Synthesis

Jing Wang, Peng Liu, Qianru Zhang, Yang Liu, Huawei Li, Xiaowei Li

提出 RKHS 框架，将 RAG（检索增强生成） 与 LLM 结合用于高层次综合（HLS）启发式优化。构建 HLS 设计案例知识库，RAG 动态检索相似历史设计；LLM 基于检索结果生成优化调度策略，替代传统手工调整的启发式规则。在 Polybench/PolyMage 基准上平均调度长度减少 11%，相比单独 LLM 提升 6.2 个百分点，资源利用率（DSP/BRAM/LUT）持平或改善。
