Agentic Verilog
Exploring the Agentic Frontier of Verilog Code Generation：首个系统评估 Agentic LLM 在 CVDP 基准上的 Verilog 生成能力，简单 Agent 包装反降性能
Patrick Yubeaton, Siddharth Garg, Chinmay Hegde · 2026-03-19 · cs.AR / cs.LG
首次系统评估 Agentic LLM 用于 Verilog 代码生成。引入多个开源硬件设计 Agent 框架（模型无关基线）。关键发现：简单 Agent 包装前沿模型反而可能降低性能（vs. 优化提示的标准前向传递）；结构化框架可匹配乃至超越非 Agent 基线；开源与闭源差距来自更高崩溃率和工具输出解释弱势。为专业化 Verilog Agent 设计指明方向。
