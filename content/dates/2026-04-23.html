NEWAgent4Chip日 ChipCraftBrain: Validation-First RTL Generation via Multi-Agent Orchestration
arXiv:2604.19856 · 2026-04-23 · RTL代码生成 / Agentic / 工业基准
提出 验证优先多智能体 RTL 生成框架，结合混合符号-神经推理与自适应多智能体编排。四项关键创新：(1) 验证第一原则替代 generate-then-fix 范式；(2) 混合符号推理减少语义错误；(3) 自适应多智能体编排降低 API 开销；(4) 针对 NVIDIA CVDP 工业基准（更难）优化。MAGE 在 VerilogEval 达到 95.9% 但在 CVDP 等工业基准上尚未经测试，ChipCraftBrain 填补这一空白并展示综合感知能力。

NEW Structural Verification for Reliable EDA Code Generation without Tool-in-the-Loop Debugging
arXiv:2604.18834 · 2026-04-21（发布）/ 2026-04-23（收录）· EDA代码生成 / LLM可靠性
LLM 已能以自然语言驱动 EDA 工作流自动化，但生成脚本的可靠执行仍是瓶颈。本文提出结构化验证方法：在执行前强制执行结构一致性，将正确性与工具交互解耦，显著提升长流程 EDA 代码生成的可靠性与效率。EDA 任务需要与分层设计数据库交互、组合工具特定 API，且需在紧密耦合的流程阶段保持一致性——本方法针对这一独特挑战设计，区别于软件领域的 LLM 验证方案。作者：Dinithi Jayasuriya, Aravind Saravanan 等，Illinois & Intel。
